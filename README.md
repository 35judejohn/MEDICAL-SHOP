# E-Medical Booking Website

This project is a simple e-medical booking website where patients can browse available doctors and book sessions.

## Features
- Browse doctor profiles
- Filter doctors by specialty
- Book an appointment with a doctor
- Simple booking request API endpoint

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```
3. Open the site at `http://localhost:5173`.

## Project Structure
- `src/` - React frontend
- `server/` - Express API backend
- `public/` - Static assets

## Notes
- The backend uses in-memory data and is ideal as a starter prototype.
- Replace placeholder assets and add persistence when scaling this project.
